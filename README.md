# Watson Toolkit

Jeremy's full skill suite — 33 skills for research, writing, dev, planning, design, and document creation.

## Install

**In Cowork:**
Settings > Add marketplace > `https://github.com/jeremyknows/watson-toolkit`

**In Claude Code:**
```
/plugin marketplace add jeremyknows/watson-toolkit
/plugin install watson-toolkit@watson-toolkit
```

See [plugins/watson-toolkit/README.md](./plugins/watson-toolkit/README.md) for the full skill catalog.
