# Iteration Guide

## Round-by-Round Focus

| Round | Focus | What to look for |
|-------|-------|-----------------|
| V1 to V2 | Structure | Hero cramped? Sections missing? Content wrong? Layout broken on mobile? |
| V2 to V3 | Content | Competitor info accurate? Pricing right? Stats sourced? ROI math correct? |
| V3 to V4 | Visual | Design elevated enough? Does it FEEL premium? Typography tight? Spacing generous? |
| V4+ | Surgical | Individual words, final pricing, commitment terms, spacing tweaks |

## Lessons from Real Builds

- V1 heroes are almost always too cramped. Default to full-width, single column.
- AI mockup images often get replaced by V3-V4 because the site itself becomes the proof. Don't over-invest in generated imagery early.
- Pricing changes every round. Don't stress V1 pricing, it will move.
- Watch for regressions. When fixing V4 issues, verify V2 fixes didn't break.
- Deployment issues (domain conflicts, old aliases, caching) eat time. Debug proactively: check for conflicting projects, clear caches, verify the URL resolves before announcing.
